Super Mario Bros

1. Project Description:

    A survival super mario game where the player controls Mario and try to get to the end.

2. Structural Plan:

    All the characters and maps will be individual objects, with draw and move functions.
    Images will be in the folder assets

3. Algorithmic Plan:

    - Acceleration, deceleration, and gravity jump
    - Platformer terrain generation for survival mode
        - Uses backtracking to see if the blocks are reachable

4. Version Control:

    Uses github to host this project, image is in this folder named "version control.png"
